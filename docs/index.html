    Kukki

    A static cookie manager that behaves like a Map.

    API Reference

    Installation

    npm install kukki

    Usage

    import { Kukki } from "kukki";

// Create a new cookie.
Kukki.set("key", "value");
Kukki.get("key"); // => "value"

// Create a persistent cookie.
const date = new Date();
date.setFullYear(date.getFullYear() + 2);
Kukki.set("persistentCookie", "persistentValue", { expires: date });
