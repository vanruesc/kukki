    public static delete(key: String, attributes: Object) source

    Removes the specified cookie.

    Cookies that have been created with a path or domain attribute can only be deleted if those attributes are provided. Cookies with an http attribute cannot be deleted programmatically.

    Params:

    NameTypeAttributeDescription
    key String

    The name of the cookie.
    attributes Object
    • optional

    Additional cookie attributes.
    attributes.path String
    • optional

    The path under which the cookie is visible. Not required if the path was never specified.
    attributes.domain String
    • optional

    The domain under which the cookie is visible. Not required if the domain was never specified.
